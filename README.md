# Overview
NOVA is an application I built to display and manage all of my finances in one place, without needing to switch apps.
It is linked to a database that stores data about accounts and the actions that have been performed on those accounts.

The app displays cards with an overview of the accounts, and a seperate frame displays statistics relating to all of them:
- Networth
- No. Accounts
- Weekly, Monthly, Quarterly, and Yearly Change
- A graph of networth over time
- A list of recently performed account actions.

Selecting an account toggles a page that displays information about the individual account:
- Account Type
- Balance
- Weekly, Monthly, Quarterly, and Yearly Change
- A graph of balance over time
- A list of recently performed account actions.
- All time interest
- All time Income and Payments
- All time net transfers

This frame also contains buttons to perform actions:
- Logging a payment
- Logging income
- Making a transfer to another account
- Logging interest
- Updating the value of investments
